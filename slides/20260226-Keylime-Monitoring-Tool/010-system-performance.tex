\begin{frame}{System Performance - Key Metrics}
  \begin{columns}[T]
    \begin{column}{0.48\textwidth}
      \begin{block}{Verifier Performance}
        \scriptsize
        Critical metrics for verifier health monitoring.
      \end{block}

      \vspace{0.1cm}
      \scriptsize

      \textbf{Resource Metrics:}
      \begin{itemize}
        \item CPU utilization per worker process
        \item Memory usage (RSS, heap)
        \item Open file descriptors
        \item Thread pool utilization
        \item Network connections (active/idle)
      \end{itemize}

      \vspace{0.1cm}

      \textbf{Application Metrics:}
      \begin{itemize}
        \item Attestations per second
        \item Queue depth (pending verifications)
        \item Worker split (web vs. verification)
        \item \texttt{dedicated\_web\_workers} utilization
        \item Exponential backoff events
        \item Rate limit rejections
      \end{itemize}
    \end{column}
    \begin{column}{0.48\textwidth}
      \begin{block}{Database Performance}
        \scriptsize
        Database health is critical for Keylime's operation.
      \end{block}

      \vspace{0.1cm}
      \scriptsize

      \textbf{Connection Pool:}
      \begin{itemize}
        \item Active/idle connections
        \item Pool size and overflow count
        \item Connection wait time
        \item Pool exhaustion events
        \item \texttt{database\_pool\_sz\_ovfl} settings
      \end{itemize}

      \vspace{0.1cm}

      \textbf{Query Performance:}
      \begin{itemize}
        \item Slow query detection (>100ms)
        \item Query count by type (SELECT/UPDATE)
        \item Table row counts (agents, attestations)
        \item Index hit ratio
        \item Migration status (Alembic revision)
      \end{itemize}
    \end{column}
  \end{columns}

  \vspace{0.2cm}

  \begin{alertblock}{Capacity Planning}
    \scriptsize
    Dashboard provides projections: at current attestation rate and agent growth,
    when will the verifier need horizontal scaling?
  \end{alertblock}
\end{frame}
